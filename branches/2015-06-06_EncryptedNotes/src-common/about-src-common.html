About the content of the folder "src-common"

In the source folder "src-common" the source code files for the library "common" ("externals\java-sdk\common") from the original Funambol open source project can be found. The classes for the following platforms of library "common" are used for ChBoSync:

  • "java" ( externals\java-sdk\common\src\main\java )
  • "java-se" ( externals\java-sdk\common\src\main\java-se )
  • "java-se-android" ( externals\java-sdk\common\src\main\java-se-android )

In former versions of ChBoSync these classes were contained in a library file: external-libs/funambol-android-common.jar .

The following two classes in package com.funambol.platform.ssl were added by the developer of the PTBV SyncML Client: RelaxedSSLSocketFactory and RelaxedSSLTrustManager. The purpose of these two classes is to enable the usage of self-signed certificates from the server.
